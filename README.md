# WeatherReport
A Weather Report based project using Spring Boot, MySQL Databse and POSTMAN API
PROJECT OVERVIEW
This project is a Spring Boot REST API for managing weather records. It allows users to perform CRUD operations (Create, Read, Update, Delete) on weather data stored in a database. The API exposes endpoints to list all weather records, retrieve a specific record, add new data, update existing entries, and delete records.
The application uses Spring Boot’s REST controller architecture along with a repository layer to interact with the database.
